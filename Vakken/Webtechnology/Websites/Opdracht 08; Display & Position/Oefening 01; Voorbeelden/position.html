Voorbeelden van Position-eigenschappen

Absolute

De eigenschap position: absolute; plaatst een element relatief ten opzichte van zijn eerstvolgende gepositioneerde (niet-statische) voorouderelement. Als er geen dergelijke voorouder is, wordt het element geplaatst ten opzichte van het initiële blok van de pagina.

Dit is een absoluut gepositioneerd element.

Fixed

De eigenschap position: fixed; plaatst een element relatief ten opzichte van het viewport. Het blijft op dezelfde plaats zelfs als de pagina wordt gescrold.

Dit is een vast gepositioneerd element.

Relative

De eigenschap position: relative; plaatst een element relatief ten opzichte van zijn normale positie. Het verplaatst het element zonder de normale stroom van de document te beïnvloeden.

Dit is een relatief gepositioneerd element.

Static

De eigenschap position: static; is de standaardpositie voor elk element. Het volgt de normale stroom van de document zonder enige speciale positionering.

Dit is een statisch gepositioneerd element.
